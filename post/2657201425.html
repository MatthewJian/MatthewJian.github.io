怎么用最麻烦的方法安装Node.js？

发表于 2022-06-29 分类于 Learning Linux 本文字数： 672 阅读时长 ≈ 1 分钟

其实在 Debian 系统中使用 apt install 就可以很轻松地安装上 Node.js，但是用 apt 安装的并不是最新版。

如果我们想自行安装最新版的 Node.js，可以去官网下载二进制文件，自行安装。

当然，这种方法最大的缺点就是——太过于麻烦！

下载 Node.js 二进制文件包

用 wget 命令下载 Node.js 二进制文件包到本地，我这里给出的是 16.15.1 LTS 版本的，你也可以在[这里][https://nodejs.org/zh-cn/download/]自己下载其他版本的二进制文件。

解压下载的安装文件

用 tar 命令解压下载的文件，在解压完成后重命名文件夹为 nodejs。

安装并应用到全局

在重命名完成后，我们把 nodejs 文件夹移动到 /usr/local/bin/ 目录下。你也可以先移动，再重命名。

进入到相关文件夹，建立软连接，将 node 和 npm 应用到全局。

验证 Node 和 npm 版本

在安装完成后，用 npm -v 和 node -v 查看 Node 和 npm 的版本。
